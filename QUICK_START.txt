╔══════════════════════════════════════════════════════════════════════════════╗
║                         IF SOMETHING GOES WRONG                               ║
╚══════════════════════════════════════════════════════════════════════════════╝

Problem: "Cannot connect to server"
Solution: 
  1. Check backend is running (Terminal #1)
  2. Check http://127.0.0.1:8000/health returns response
  3. Verify no port conflicts (port 8000 and 3000 must be free)

Problem: "Invalid email or password"
Solution:
  1. Use default account: mentor@example.com / Password123
  2. Check caps lock is OFF
  3. Check password is exactly: Password123
  4. Or create new account via sign up

Problem: Login appears to work but no redirect
Solution:
  1. Check browser console (F12) for errors
  2. Check backend console for error messages
  3. Clear localStorage: Press F12 → Application → Local Storage → Clear All
  4. Reload page and try again

Problem: Frontend won't start
Solution:
  1. Delete node_modules: rm -r d:\pbl\frontend\node_modules
  2. Delete package-lock.json: rm d:\pbl\frontend\package-lock.json
  3. Run again: npm install && npm run dev
